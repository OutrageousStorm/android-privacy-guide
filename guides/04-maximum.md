# Level 4 — Maximum Privacy (GrapheneOS)

The highest achievable security/privacy. Requires GrapheneOS (Pixel-only).

## Why GrapheneOS

- Hardened memory allocator
- Verified boot + locked bootloader
- Per-app privacy controls
- Sandboxed Google Play
- No telemetry

## Install

https://grapheneos.org/install/web

## Setup

Settings → Security:
- Auto-reboot: 8 hours
- Storage Scopes: per-app storage access
- Network: per-app toggles

Recommended apps: Vanadium (browser), Molly (Signal), FairEmail, Organic Maps, Aegis

## Checklist

✅ Bootloader locked
✅ Per-app permissions revoked
✅ F-Droid primary
✅ Private DNS set
✅ Auto-reboot enabled
